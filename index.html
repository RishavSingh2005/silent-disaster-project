Live weather integration

Rainfall and temperature data is fetched from the Open-Meteo API in real time — the same source used by meteorological research tools. No simulations. If a city is experiencing unusually heavy rain right now, this system will show it.

Transparent risk scoring

Every score is built from four auditable rules: rainfall acceleration, cumulative volume, historical flood sensitivity, and temperature-rainfall interaction. The logic is fully visible in the Data Transparency panel. No black box.

Alert & notification system

Add cities to your watchlist. When assessed risk crosses your threshold, the system sends an in-app alert and a browser notification. Alert history is stored locally across sessions.
